About Web-Site


How to use this website

This website allows to add items for selling and buy already placed items. You can add an item for it to be sold on the page 'Add an item'. An item also can be deleted on the page 'Delete an item'. This website also has plugged "Fondy" payment system (Test payment).

How it works

This web-site is written by using Flask framework. User can add or delete items from the Sqlite database (created with SQLAlchemy), then records would show on the main page. Also Jinja Template is used in HTML pages.
